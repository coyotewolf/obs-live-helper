First Launch Setup

第一次使用設定

Spotify Client ID 與 Discord StreamKit Overlay URL 可以之後再設定。未設定時，控制台仍可使用；Spotify 相關功能會暫時停用，Discord 頭像頁會先使用內建範例設定。

正在檢查設定...
01 設定 Spotify Client ID

OBS Live Helper 需要連到使用者自己的 Spotify Developer App。先建立 App，再把 Client ID 貼到下方。

開啟 Spotify Developer Dashboard
02 加入 Spotify Redirect URI

在 Spotify App 的 Settings 裡，把下面這個網址加到 Redirect URIs。

http://127.0.0.1:5172/api/spotify/callback
03 貼上 Spotify Client ID

在 Spotify App Settings 找到 Client ID，貼到下方欄位。

04 設定 Discord StreamKit Overlay URL

到 Discord StreamKit 產生自己的 Voice Widget 網址，或先貼入目前 OBS 可用的 StreamKit URL。此功能用於 dcprofilepic.html。

開啟 Discord StreamKit
05 進入控制台並登入 Spotify

可以先前往控制台使用基本功能。若之後要啟用 Spotify，再回來填寫 Client ID 並登入 / 重新授權 Spotify。

前往控制台
設定流程補充
  1. Spotify Redirect URI 必須填：http://127.0.0.1:5172/api/spotify/callback
  2. Discord StreamKit URL 必須是 https://streamkit.discord.com/overlay/voice/... 開頭。
  3. 如果 Discord 頭貼 Overlay 沒有顯示，請確認 Discord 桌面版已開啟、已登入，並且 StreamKit 原始網址在瀏覽器中可用。
  4. 若要換帳號或換 Discord 頻道，可以回到這個設定頁重新儲存。
