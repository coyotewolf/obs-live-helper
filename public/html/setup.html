First Launch Setup

先設定 Spotify Client ID

OBS Live Helper 需要連到使用者自己的 Spotify App。這個設定只會存放在本機，不會上傳到雲端。

正在檢查設定...
01 開啟 Spotify Developer Dashboard

登入 Spotify 後建立一個 App，名稱可以填 OBS Live Helper。

開啟 Spotify Developer Dashboard
02 加入 Redirect URI

在 Spotify App 的 Settings 裡，把下面這個網址加到 Redirect URIs。

http://127.0.0.1:5172/api/spotify/callback
03 貼上 Client ID

在 Spotify App Settings 找到 Client ID，貼到下方欄位後儲存。

找不到 Client ID 或 Redirect URI 設定？
  1. 進入 Spotify Developer Dashboard。
  2. 點 Create app。
  3. App name 可填 OBS Live Helper。
  4. Redirect URI 必須填：http://127.0.0.1:5172/api/spotify/callback
  5. 建立後進入 App Settings，複製 Client ID。
