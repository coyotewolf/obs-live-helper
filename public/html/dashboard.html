OBS Live Helper

直播 Overlay 控制台

集中管理 Spotify 歌詞、目前播放、佇列清單與自訂文字公告。先登入 Spotify，再把需要的 Overlay URL 加到 OBS Browser Source。

預覽歌詞頁
Quick Start

第一次使用照這樣做

01 啟動 Helper

在專案資料夾執行 npm start，保持終端機視窗開著。

02 登入 Spotify

按上方登入按鈕。若改過 scope、token 過期或 queue 抓不到，就重新授權一次。

03 加入 OBS

在 OBS 新增 Browser Source，貼上下方需要的 Overlay URL。

04 播放音樂

Spotify 需要有 active device，歌詞、目前播放與佇列才會正常更新。

使用提醒
  • OBS 沒更新時，在 Browser Source 按 Refresh cache。
  • 如果 queue 空白，確認 Spotify 正在播放並重新授權。
  • 如果歌詞顯示「無歌詞」，通常是 LRCLib 找不到同步歌詞。
