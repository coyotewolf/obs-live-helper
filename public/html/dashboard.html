OBS Live Helper

直播 Overlay 控制台

集中管理 Spotify 歌詞、目前播放、佇列清單、自訂文字公告、觀眾 QR Code 點歌、小目標卡片與直播時鐘。先登入 Spotify，再把需要的 Overlay URL 加到 OBS Browser Source。

Quick Start

第一次使用照這樣做

01 啟動 Helper

在專案資料夾執行 npm start，保持終端機視窗開著。

02 登入 Spotify

按上方登入按鈕。若改過 scope、token 過期或 queue 抓不到，就重新授權一次。

03 加入 OBS

在 OBS 新增 Browser Source，貼上下方需要的 Overlay URL。

04 開放點歌

安裝 cloudflared 後可開外網 Tunnel，讓直播觀眾掃 QR Code 點歌。

安全提醒
  • Dashboard 請盡量只從 127.0.0.1 本機開啟；觀眾只需要掃 request.html 的 QR Code。
