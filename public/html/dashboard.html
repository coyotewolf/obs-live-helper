OBS Live Helper

直播 Overlay 控制台

集中管理 Spotify 歌詞、目前播放、佇列清單、自訂文字公告、觀眾 QR Code 點歌、小目標卡片與直播時鐘。先登入 Spotify，再把需要的 Overlay URL 加到 OBS Browser Source。

Quick Start第一次使用照這樣做 展開 / 收合
01 啟動 Helper

在專案資料夾執行 npm start，保持終端機視窗開著。

02 登入 Spotify

按上方登入按鈕。若改過 scope、token 過期或 queue 抓不到，就重新授權一次。

03 加入 OBS

在 OBS 新增 Browser Source，貼上下方需要的 Overlay URL。

04 開放點歌

安裝 cloudflared 後可開外網 Tunnel，讓直播觀眾掃 QR Code 點歌。

Backup備份與資料保護 展開 / 收合

匯出目前 OBS Live Helper 的本機設定與 runtime 資料，包含 Spotify / Discord 設定、自訂文字樣式、Overlay 設定、觀眾點歌設定與上傳字型。備份檔會下載成 JSON；歌詞 log 與 LRCLib 快取不會放進備份，避免檔案過大。

安全提醒
  • Dashboard 請盡量只從 127.0.0.1 本機開啟；觀眾只需要掃 request.html 的 QR Code。
  • 外網 Tunnel 會把本機服務公開出去，請保留冷卻時間，並謹慎開放播放控制與切歌。
  • 請求插播不會自動操作 Spotify；主播同意後需自行在 Spotify 播放。
  • 給別人使用時，請讓對方用自己的 Spotify Developer App 與 Spotify 帳號登入。
