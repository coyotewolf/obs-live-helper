OBS Live Helper

直播 Overlay 控制台

集中管理 Spotify 歌詞、目前播放、佇列清單與自訂文字畫面。先登入 Spotify，再把需要的 Overlay URL 加到 OBS Browser Source。

Quick Start

第一次使用照這樣做

01

啟動 Helper

在專案資料夾執行 npm start，保持這個視窗開著。

02

登入 Spotify

按上方登入按鈕。如果修改過 scope 或 token 過期，就重新授權一次。

03

加入 OBS

複製下方 Overlay URL，在 OBS 新增 Browser Source 貼上。

04

播放音樂

Spotify 必須有正在播放的裝置，歌詞與佇列才會正常更新。

        Guide
        
OBS 建議設定

      


      

        

          
Browser Source 尺寸

          
歌詞頁可用 1920×360 或 1920×520；目前播放建議 900×240；佇列建議 650×520。

        

        

          
透明背景

          
歌詞、目前播放、佇列頁面都已設計成透明背景；自訂文字頁可在工具列勾選「透明」。

        

        

          
常見問題

          
如果歌詞顯示「無歌詞」，通常是 LRCLib 找不到同步歌詞；如果 queue 空白，請確認 Spotify 正在播放且已重新授權。
